## Datasets: Folder Structure and Preparation

In the following, we present the folder structure required to use the different datasets with **Hy²DL-student**. Currently, only CAMELS\_DE is implemented. 

### CAMELS\_DE

The CAMELS\_DE dataset can be downloaded from [Zenodo](https://doi.org/10.5281/zenodo.16755906).
To add this dataset to the library, use the following folder structure:

```
CAMELS\_DE/
  timeseries/ 
  CAMELS\_DE\_climatic\_attributes.csv
  CAMELS\_DE\_humaninfluence\_attributes.csv
  ...
  CAMELS\_DE\_xxx\_attributes.csv
```
